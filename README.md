# آتیه | پنل مدیریت استخدام و امور اداری

پنل وب حرفه‌ای، مدرن و کاملاً کاربرپسند برای مدیریت منابع انسانی، استخدام و امور اداری.

**ویژگی‌های فعلی (v0.1):**
- رابط کاربری زیبا با Tailwind CSS v3 (تم تاریک حرفه‌ای + RTL کامل فارسی)
- منوی کناری کامل + داشبورد تعاملی
- **تقویم و تاریخ کامل شمسی (جلالی)** با تبدیل خودکار
- نمایش **زنده ساعت و تاریخ روز** در هدر داشبورد
- افزودن کارمند جدید با فرم تعاملی + جدول استخدام‌های اخیر (ذخیره در حافظه مرورگر)
- طراحی responsive و آماده برای Tauri (تبدیل به .exe دسکتاپ)

**ساختار پروژه:**
- `index.html` → پنل کامل (همه چیز در یک فایل برای شروع سریع)

**نحوه اجرا (فعلی):**
1. فایل `index.html` را در مرورگر باز کنید.
2. برای توسعه محلی: از Live Server یا هر وب سرور ساده استفاده کنید.

**مراحل بعدی (طبق دستور شما):**
- تکمیل بخش‌های منو (کارکنان کامل، جذب نیرو، حضور و غیاب، حقوق و دستمزد، گزارش‌ها)
- افزودن تقویم شمسی پیشرفته
- ذخیره‌سازی واقعی (localStorage / IndexedDB)
- راه‌اندازی پروژه Tauri برای خروجی .exe
- اتصال به API در صورت نیاز

همه تغییرات به صورت کامل در گیت‌هاب پوش می‌شود.
